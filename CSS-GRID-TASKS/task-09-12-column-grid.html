Master the industry-standard 12-column grid system using repeat(12, 1fr).

Box 1 (Span 4)

Why 12 columns?

12 is divisible by 2, 3, 4, and 6, making it extremely versatile for creating complex layouts.

Box 2 (Span 8)

How to define it?

Use grid-template-columns: repeat(12, 1fr); on the parent container.

Box 3 (Span 6)

What is a col-span?

Items span a specific number of tracks using grid-column: span [number];.

Box 4 (Span 6)

Standard layouts?

Common layouts include 4-col (span 3), 3-col (span 4), and 2-col (span 6) structures.

Box 5 (Full Width Span 12)

Can we nest grids?

Yes! A grid item can also be a grid container, allowing for deeply nested and complex designs.
