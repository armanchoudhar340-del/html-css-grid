Answers

1. Why is 12 the "magic number" for grid systems?

Because 12 is divisible by many numbers: 2, 3, 4, and 6. This makes it very easy to create layouts that are 1/2, 1/3, 1/4, or even 1/6 of the width.

2. How do you create 12 columns in CSS?

Using grid-template-columns: repeat(12, 1fr);. This divides the space into 12 equal vertical slices.

3. How do you make an item span 3 columns?

By using grid-column: span 3;. This is a shorthand that tells the item to cover the next 3 available columns.
