Answers

1. What does the 'gap' property do?

The gap property creates empty space between grid items (rows and columns). It does NOT add space around the outside of the container.

2. What is the difference between 'gap' and 'margin'?

margin adds space outside of an element's border. gap is applied to the container and only adds space between the children (gutters), making it more consistent for layouts.

3. Can you have different gaps for rows and columns?

Yes! You can use row-gap: 10px; and column-gap: 30px; or the shorthand gap: 10px 30px; (row column).
