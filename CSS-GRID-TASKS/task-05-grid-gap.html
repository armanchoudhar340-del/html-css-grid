Learn how to add space between grid items using the gap property.

Box 1

What does gap do?

It adds space (gutters) between grid items, making the layout cleaner.

Box 2

Can we set row gap only?

Yes, row-gap adds space only between the horizontal rows.

Box 3

Can we set column gap only?

Yes, column-gap adds space only between the vertical columns.

Box 4

What is the shorthand?

gap: 20px 10px; sets row gap to 20px and column gap to 10px.

Box 5

Does gap add outer space?

No, gap only adds space BETWEEN items, not on the outer edges of the container.

Box 6

Observation:

The items are now separated by clear spacing, improving readability and design.
