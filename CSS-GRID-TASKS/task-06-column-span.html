Answers

1. What does 'grid-column: 1 / 3' mean?

It means the item starts at grid line 1 and ends at grid line 3. This makes it cover 2 columns.

2. How do you make an item span all columns?

You can use grid-column: 1 / -1;. The -1 represents the last grid line, no matter how many columns you have.

3. How do the other items react when one spans?

CSS Grid automatically pushes the other items to the next available slot in the grid.
