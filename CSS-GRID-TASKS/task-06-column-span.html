Learn how to make grid items span multiple columns using grid-column.

Box 1 (Spanned)

What does span mean?

It allows this item to cover multiple columns. This box uses grid-column: span 2;.

Box 2

How to span 2 columns?

Use grid-column: span 2; or 1 / 3; to define the start and end lines.

Box 3

Row spanning?

Yes! You can also use grid-row: span 2; to make an item cover multiple rows.

Box 4

What is 1 / -1?

grid-column: 1 / -1; makes an item span the entire width of the grid container.

Box 5

Auto-flow?

Other grid items automatically move to the next available space around spanned items.

Box 6

Observation:

Box 1 is wider because it occupies two column tracks, while others occupy only one.
