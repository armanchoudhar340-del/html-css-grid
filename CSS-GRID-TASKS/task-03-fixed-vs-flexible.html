Compare how px (Fixed) and fr (Flexible) units behave in a single grid.

Box 1

What is a fixed column?

A column with a set width, like 200px, that does not change size when the window is resized.

Box 2

What is a flexible column?

A column that uses fr units to take up a fraction of the remaining space in the container.

Box 3

What happens with mixed units?

In 200px 1fr 1fr, the first column stays at 200px, and the rest of the space is split equally.

Box 4

Why use fixed columns?

They are useful for sidebars or elements that require a specific, unchanging width.

Box 5

Can we mix units?

Yes! You can combine px, %, and fr in your grid-template definitions.

Box 6

Observation:

The flexible columns adjust while the fixed one remains constant as you resize the browser.
