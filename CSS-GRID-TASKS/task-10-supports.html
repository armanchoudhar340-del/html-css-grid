Learn how to use @supports for browser compatibility and progressive enhancement.

Box 1

What is @supports?

A CSS feature query rule that checks if a browser supports a specific CSS property/value.

Box 2

Why use it for Grid?

It allows you to provide a fallback layout (like Flexbox or Floats) for older browsers.

Box 3

How to write it?

@supports (display: grid) { ... } wraps your grid-specific styles safely.

Box 4

Progressive Enhancement?

Building a basic layout first, then adding Grid as an "upgrade" for modern browsers.

Box 5

Is it still necessary?

Grid support is now excellent, but @supports is vital for newer features like Subgrid.

Box 6

Observation:

If you see these boxes in columns, your browser passed the @supports query!
