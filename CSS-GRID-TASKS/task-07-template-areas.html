Define your layout visually using grid-template-areas.

Box 1 (Header)

What are named areas?

They allow you to name sections of your grid and place items into them visually.

Box 2 (Sidebar)

How to name an area?

Use grid-area: header; on the item, then define it in the container.

Box 3 (Content)

What is the benefit?

It creates a highly readable ASCII-like layout in your CSS file.

Box 4 (Footer)

What is a dot (.) in areas?

A dot represents an empty cell in the grid-template-areas string.

Box 5

Spanning in areas?

Items span multiple cells by repeating the area name in the template string.

Box 6

Observation:

The layout is controlled by names like 'header', 'sidebar', and 'footer' instead of numbers.
