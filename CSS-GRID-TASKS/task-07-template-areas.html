Answers

1. What is grid-template-areas?

It's a visual way to define the layout by giving names to sections of the grid and mapping them where they should appear.

2. How do you make an area span multiple cells?

By repeating the area name in the grid-template-areas string. For example: "header header header" makes the header span 3 columns.

3. Why is this useful for rearranging layouts?

Because you can change the entire structure for mobile or desktop just by changing the grid-template-areas string in a media query, without changing the HTML.
