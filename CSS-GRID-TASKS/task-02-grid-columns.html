Answers

1. What does '1fr' mean?

'fr' stands for 'fractional unit'. 1fr means one part of the available space. 1fr 1fr 1fr divides the grid into 3 equal parts.

2. How do the boxes behave when you resize the window?

Because we used fr units, the columns automatically grow or shrink to fill the window while always staying in 3 equal columns.

3. How do you create 4 columns instead of 3?

You would change the CSS property to: grid-template-columns: 1fr 1fr 1fr 1fr;
