Learn how to define specific columns using grid-template-columns.

Box 1

What does fr stand for?

fr means fractional unit. It divides the available space into parts so each column gets equal space.

Box 2

How many columns are created?

Three columns are created because three fr values are used in grid-template-columns.

Box 3

What happens when the screen size is reduced?

The columns shrink automatically and adjust according to the screen size.

Box 4

Do items overflow or adjust?

The items adjust their width and stay inside the container instead of overflowing.

Box 5

Why is fr better than px for layouts?

fr is flexible and responsive, while px is fixed and does not adjust well on different screens.

Box 6

Observation:

The boxes are arranged in three columns and resize smoothly when the browser window is resized.
