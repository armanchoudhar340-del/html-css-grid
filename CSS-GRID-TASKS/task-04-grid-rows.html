Learn how to control the height of rows using grid-template-rows.

Box 1

What does grid-template-rows do?

It defines the height of the rows in the grid layout.

Box 2

How many rows are here?

Two rows are explicitly defined in this example (e.g., 200px and 150px).

Box 3

Content overflow?

If content is larger than the fixed height, it might overflow unless overflow is handled.

Box 4

What is min-content?

It sets the row height to the minimum size possible without causing overflow.

Box 5

What is max-content?

It sets the row height to perfectly fit the largest piece of content in that row.

Box 6

Observation:

Rows can have different heights independently, allowing for structured vertical sections.
