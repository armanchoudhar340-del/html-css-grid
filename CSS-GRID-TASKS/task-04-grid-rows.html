Answers

1. What does 'auto' do in grid-template-rows?

It makes the row height expand to fit the content inside it. If you have more text, the row gets taller.

2. How do you set all rows to be 150px?

You can use: grid-template-rows: 150px 150px 150px; or grid-template-rows: repeat(3, 150px);

3. What is the difference between grid-template-columns and grid-template-rows?

Columns define the horizontal space (width), while rows define the vertical space (height).
