In this task, we will initialize a grid container using display: grid.

Box 1

What does display: grid do?

It turns the element into a grid container, enabling grid layout for its direct children.

Box 2

Why are the items stacked?

By default, a grid creates a single column, so items stack vertically like normal block elements.

Box 3

What is a Grid Container?

The parent element that has display: grid applied. It defines the grid context.

Box 4

What is a Grid Item?

Any direct child of a grid container automatically becomes a grid item.

Box 5

Can we use inline-grid?

Yes! display: inline-grid makes the container behave like an inline element instead of a block.

Box 6

Observation:

Even without columns defined, the items now respect the grid container's padding and gap properties.
