Answers

1. Why does the layout look the same as a normal block layout?

When you only use display: grid without defining columns or rows, the grid defaults to a single column that spans the full width, just like normal block elements (divs).

2. What is a Grid Container?

The parent element that has display: grid applied to it. It holds all the grid items.

3. What is a Grid Item?

The direct child elements of the Grid Container. In this demo, the 6 divs inside are the grid items.
